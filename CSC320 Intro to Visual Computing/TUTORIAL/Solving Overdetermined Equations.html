Solving Overdetermined Equations

Given $\vec b_{n\times 1}, A_{m\times n}$, to solve $\vec x_{m\times 1}, A\vec x = \vec b$

$$\begin{align*}A^T \vec b &= A^T A \vec x \\ (A^T A)^{-1}A^T \vec b &= \vec x\end{align*}$$

$(A^TA)^{-1}A$ is called pseudo inverse, which can be computed by SVD

Singular Value Decomposition wiki¶

For a matrix $A$, we can decompose $A_{m\times n} = U_{m\times m}\Sigma_{m\times n} V^T_{n\times n}$ where $\Sigma$ is diagonal with non-negative $\sigma$'s, $U,V$ are unary, i.e. $U^T = U^{-1}, V^T = V^{-1}$

Claim $A^+ := (A^T A)^{-1}A^T = V(\Sigma^T \Sigma)^{-1}\Sigma U^T$

proof. $$\begin{align*} (A^T A)^{-1}A^T &= (V\Sigma^T U^T U\Sigma V^T)^{-1}V\Sigma^T U^T\\ &= (V^{-1})^T\Sigma^{-1}U^{-1}(U^{-1})^T(\Sigma^{-1})^TV^{-1}V\Sigma^T U^T \\ &= V\Sigma^{-1}U^{-1}U(\Sigma^{-1})^T V^{-1}V\Sigma^T U^T &U,V\text{ unary} \\ &= V\Sigma^{-1} I (\Sigma^{-1})^TI\Sigma^T U^T \\ &= V\Sigma^{-1} U^T &\text{since }\Sigma \text{ is diagonal} \end{align*}$$
