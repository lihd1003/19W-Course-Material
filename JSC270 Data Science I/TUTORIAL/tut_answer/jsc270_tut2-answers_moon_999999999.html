# code for question 1 (b)
# read wiki page from the web store the table
wiki_page = 'https://en.wikipedia.org/wiki/List_of_regions_of_Canada'
df = pd.read_html(
    wiki_page, 
    header=0, # since the html does not tag (<th>) 
              # the first row as column, 
              # you need to manually specify it
    attrs = {"class":"wikitable"} # when you know specific html attributes 
                                  # of the table you want you can specify 
                                  # them here for a more specific query
)
# read the six region columns and prov/territories
regions = df[0][[
    'All provinces and territories', 
    'Six-region model[1]'
]]

regions = regions.fillna(
    # propogate non-null values forward
    method = 'ffill', axis = 0 
).rename(columns={
    # rename columns
    "All provinces and territories": "prov", 
    "Six-region model[1]": "region"
}).replace(
    # rename the province name to match the other table
    ['Newfoundland and Labrador'], ['Newfoundland']
)

# Change province names so they match in both data sets
# `regex=True` allows you to use regular expressions
fludat = (
    fludat
    .replace('Province of', '', regex = True)
    .replace('Québec', 'Quebec', regex = True)
    .rename(columns={"Reporting Laboratory": "prov"})
)

# Alberta, Ontario, Quebec, Sask have extra white space in their names so remove
fludat['prov'] = fludat['prov'].str.strip()

# Merge regions with flu data and only keep required columns
fludat1 = pd.merge(
    fludat, regions, on = 'prov'
).iloc[0:,[0,1,2,3,4,5,23]]
# change the data type to float(numeric) instead of object(string)
fludat1['Flu Tested'] = fludat1['Flu Tested'].astype(float)

# create Proportion of Flu A Positive 
# (defined as the ratio of Total Flu A Positive to Flu Tested)
fludat1['propfluA'] = fludat1.apply(
    lambda x: x['Total Flu A Positive']/x['Flu Tested'],
    axis=1
)
fludat1.round(3)

A categorical variable is one for which the measurement scale consists of a set of categories. Categorical variables for which levels do not have a natural ordering a called nominal variables. Categorical variables that have ordered levels are called ordinal variables. Categorical variables that have numerical distances between two levels of the scale are called often called interval variables.

Categorical variables can attain a discrete number of values while a continuous variable can attain values that are a subset of the set of real numbers.

Answer for Question 4¶

d) The null hypothesis ($H_0$) is that the proportions of influenza A subtypes are independent of provinces and territories and thus all provinces and territories expect equal proportions. The alternative hypothesis ($H_1$) is that the proportions are not independent of the provinces and territories and thus the expected frequencies differ by provinces and territories.

e) The p-value is $0.0<0.05$ and thus the null hypothesis is rejected. We can conclude that the the proportions are note independent of the provinces and territories at $95%$ confidence level.

Notes on validity of the test:¶

  • The provinces are probably not independent to begin with. Neighbouring provinces/territories are probably more likely to show more similarities compared to regions that are far apart due to climate, terrain, etc.
  • From the chi2_contingency documentation,

    An often quoted guideline for the validity of this calculation is that the test should be used only if the observed and expected frequency in each cell is at least 5.
